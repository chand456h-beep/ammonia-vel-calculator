# Ammonia VLE Calculator

## Overview

A Python-based engineering tool for calculating ammonia vapor-liquid equilibrium (VLE) properties from pressure and temperature inputs.

This project is intended for:

- Process Engineers
- DCS Engineers
- Ammonia Plant Operators
- Production Engineers
- Chemical Engineering Students

---

## Features

### Current Version

- Pressure input (kg/cm²G)
- Temperature input (°C)
- Phase determination
- Saturation pressure calculation
- Saturation temperature calculation
- Ammonia vapor-liquid equilibrium analysis

### Planned Features

- NH3 property package
- Ammonia density calculation
- Vapor fraction estimation
- Receiver inventory estimation
- Storage tank calculations
- Relief valve engineering tools
- Process trend analysis
